Wakanda Widgets Part 2 - File Upload




The File Upload widget allows you to asynchronously upload one or more files from your Page to the Web Server.

Wakanda Widgets Part 2 - File Upload


Two Options for Where the Uploaded File is Stored


  • A Folder in the DataFolder of your Project
    (file can be any type)
  • Directly to the Wakanda Datastore
    (png, jpg, bmp, tiff, fle, jif, rle, dib, gif, emf, svg, or pdf)
