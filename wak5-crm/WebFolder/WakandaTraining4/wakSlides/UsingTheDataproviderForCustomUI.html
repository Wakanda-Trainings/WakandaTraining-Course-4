Dataprovider and Custom User Interfaces


Steps For Creating Custom User Interfaces with the Dataprovider




  1. Store your HTML snippets in Javascript Templates.
  2. Use the Dataprovider to supply data and render these templates.
  3. Create a custom CSS file for your page to apply style.
  4. Use jQuery to create event handlers for your custom UI.

Dataprovider and Javascript Templates


It’s considered a best practice to keep your HTML, CSS, and JavaScript separate




When you need to build a custom HTML interface use Javascript Templates to keep your HTML, CSS and Javascript separate.


This makes it easier to keep track of the code and spot errors, when necessary.
